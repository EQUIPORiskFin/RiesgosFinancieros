{\bf Modelo mixtos de Bernoulli}

\noindent {\bf Definición.} Dada alguna $p<m$ y un vector aleatorio de dimensión $p$, $\Psi=(\Psi_1,\ldots,\Psi_p)^\top$, el vector aleatorio $Y=(Y_1,\ldots,Y_m)^\top$ es un modelo mixto de Bernoulli con el vector de factores $\Psi$ si existen funciones $p_i\colon \R^p\to[0,1]$, $1\leq i\leq m$, tal que la probabilidad condicional sobre $\Psi$ de $Y$ variables aleatorias independientes bernoulli que satisfacen $P(Y_i=1|\Psi=\psi)=p_i(\Psi)$.

\noindent Para $y=(y_1,\ldots,y_m)^\top\in\{0,1\}^m$ tenemos
$$P(Y=y|\Psi=\psi)=\prod_{i=1}^mp_i(\Psi)^{y_i}(1-p_i(\Psi))^{1-y_i},$$
y la distribución no condicional de $Y$ se obtiene de integrar esta función sobre el vector de factores $\Psi$. En particular, la probabilidad de incumplimiento del acreditado $i$ es $\bar p_i=P(y_i)=E(p_i(\Psi)$.

Los modelos mixtos de Bernoulli de un factor se presentan cuando $\Psi$ es una variable aleatoria con valores en $\R$ y $p_i(\psi)\colon \R \to [0,1]$ son funciones tales que, condicionadas en $\Psi$ el vector indicador de default $Y$ es un vector de variables aleatorias Bernoulli.
Una simplificación ocurre cuando las funciones $p_i$  son idénticas para todo $i$, denotemos por $Q:=p_1(\Psi)$, y sea $G(q)$ la función de distribución de esta variable mixta.

{\bf Distribución mixta Probit-normal}. Aquí $Q=\Phi(\mu+\sigma\Psi)$ para $\Psi\sim N(0,1)$, $\mu\in \R$ y $\sigma>0$, donde $\Phi$ es la distribución normal estándar. El modelo de KMV se puede ver como un modelo mixto Probit-Normal de un factor.

Para estimar el VaR se estiman las pérdidas y ganancias del portafolio y se ordenan, como se muestra a continuación:
\item Se construye la serie de rendimientos. Es decir, se estiman las variaciones diarias de los factores de riesgo:
\begin{displaymath} R_{t-1}=T_j\Big(\frac{FR_t}{FR_{t-1}}\Big)\quad j=1,2 \end{displaymath}
donde $T_1(x)=\ln(x)$, $T_2(x)=x-1$.
\item Se estima la serie alternativa del factor de riesgo. Para ello, al valor actual del factor de riesgo se agrega el valor de las variaciones calculadas:
\begin{displaymath}
FR_0\cdot T_j^{-1}\left(\begin{array}{c}
R_1\\
R_2\\
\vdots\\
R_n	
\end{array}\right)=\left(\begin{array}{c}
FR_{01}\\
FR_{02}\\
\vdots\\
FR_{0n}	
\end{array}\right)
\end{displaymath}
\item El portafolio se revalúa con cada uno de los valores estimados de los factores de riesgo.

\item Se calculan las pérdidas y ganancias del portafolio. Estas se obtienen de la diferencia entre el valor del portafolio estimado con cada uno de los escenarios, y el valor del portafolio vigente en la fecha de valuación.

\item Se calcula el VaR con base en el nivel de confianza (percentil o cuantil) elegido.
\end{enumerate}\\*
